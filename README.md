# 🌍 Project Overview

The **Enterprise Institute Management System (ERP) with Learning Management System (LMS)** is a comprehensive web-based platform designed to automate and simplify every major operation of an educational institute.

Unlike traditional institute software that only manages student records, this platform combines:

- Institute ERP
- Learning Management System (LMS)
- Student Portal
- Teacher Management
- Academic Management
- E-Commerce Store
- Certificate Center
- Blog CMS
- Tutorial Platform
- Support Ticket System
- Discussion Forum
- User & Staff Management

into a single integrated solution.

The system enables educational organizations to efficiently manage students, teachers, live classes, recorded courses, assignments, certificates, accounts, products, support requests, blogs, tutorials, and much more through one centralized dashboard.

It is built using modern Laravel architecture and provides a fast, scalable, secure, and user-friendly experience.

---

# 🎯 Why This Project?

Managing an educational institute manually becomes increasingly difficult as the number of students, teachers, courses, and administrative tasks grows.

Many institutes rely on multiple disconnected systems:

- Excel Sheets
- WhatsApp Groups
- Google Drive
- Paper Records
- Separate Accounting Software
- Third-party LMS Platforms

This project eliminates those problems by providing one centralized platform where administrators can control every aspect of the institute.

The result is:

- Faster operations
- Better communication
- Improved reporting
- Reduced paperwork
- Higher productivity
- Better student experience

# ⚡ Nginx Configuration

```nginx
server {

    listen 80;

    server_name your-domain.com;

    root /var/www/project/public;

    index index.php;

    location / {

        try_files $uri $uri/ /index.php?$query_string;

    }

    location ~ \.php$ {

        include fastcgi_params;

        fastcgi_pass unix:/run/php/php8.3-fpm.sock;

        fastcgi_param SCRIPT_FILENAME $document_root$fastcgi_script_name;

    }

}
```
